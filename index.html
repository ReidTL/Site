Projects

Brief description of some projects and key assignments.
These offer some insight into my skills and abilities

Object Based Image Analysis: Powerful approach to analyzing satellite imagery that focuses on identifying and characterizing meaningful objects or features within the image.
Spatial Databases: Working with SQL on spatial data to carry out several spatial based queries. This leans on the Open source architecture of PostGIS and QGIS
Google Earth Engine: Cloud-based platform for geospatial analysis. I use it to access and analyze massive datasets of satellite imagery and other geospatial data using a powerful scripting language. 
Timeseries analysis of Landsat data with GEE: Various analyses included land surface temperature, Vegetation change, Urban growth over different areas of interest.
EO browser - Custom Scripts: Online tool that, allows one to visualize and explore satellite imagery. Adding Custom scripts enables powerful visualizations that are repeatable over any area.
RADAR data analysis with SNAP: Toolbox designed specifically for processing and analyzing RADAR imagery e.g. InSAR, Before/After analyses, etc.
Sen2Cube.at: Semantic Earth Observation Data Cube Analysis. A semantic EO data cube is a data cube, where for each observation at least one nominal (i.e., categorical) interpretation is available and can be queried in the same instance
QGIS + Py. Leveraging Python and Qgis to automate routine workflows.
